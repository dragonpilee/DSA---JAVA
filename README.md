## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/dragonpilee/DSA---JAVA.git
   cd DSA---JAVA
   ```

2. **Compile and run any Java file**
   ```bash
   javac FileName.java
   java FileName
   ```
